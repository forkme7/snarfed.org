IndieWeb Press This bookmarklets for WordPress

One big _raison d'Ãªtre_ of the [IndieWeb](http://indiewebcamp.com/) movement is
using your own web site to [like](http://indiewebcamp.com/like),
[repost](http://indiewebcamp.com/repost),
, and [RSVP](http://indiewebcamp.com/rsvp)
to posts and events posted anywhere on the web. You do this by annotating links
on your site with simple
[microformats2](http://microformats.org/wiki/microformats2) HTML.

Having said that, most people don't want to write HTML just to comment or like
something. WordPress's
[Press This bookmarklets](http://codex.wordpress.org/Press_This) can already
start a new post with a link to the page you're currently viewing. This code
adds IndieWeb microformats2 markup to that link. Combined the
[wordpress-webmention](https://github.com/pfefferle/wordpress-webmention)
plugin, you can use this to reply to, like, repost, or RSVP to the current page
with just two clicks.

What's more, if you're currently on a Facebook post or Twitter tweet, this adds
the [Bridgy Publish](https://www.brid.gy/about#publish) link that will reply,
like, favorite, retweet, or even RSVP _inside_ those social networks.

Here's how to set it up. First, [download `press_this.js`]() and put it in your
theme directory. Then, add this to your theme's `functions.php` file.

    function snarfed_add_press_this_js() {
    	wp_enqueue_script('snarfed-press-this',
    	                  get_stylesheet_directory_uri() . '/press_this.js',
    	                  false, '1.0', true);
    }
    add_action('admin_print_scripts-press-this.php', 'snarfed_add_press_this_js');

Finally, create bookmarks in your browser for these bookmarklets, then edit them
and replace `YOUR_SITE` with your site's domain.
