[PyBlosxom](http://pyblosxom.sf.net/), the server software that runs this site,
uses [flavours](http://pyblosxom.sf.net/1.3.1/manual/c224.html) to render pages
in different formats. Pyblosxom comes with a number of flavours, including
plain HTML, RSS (0.91 and 2.0), and Atom.

I've added an [RDF flavour](/space/rdf_flavour.tar.gz) that uses
[Dublin Core](http://dublincore.org/schemas/xmls/) elements.
For an example, check out the
[RDF for this page](pyblosxom+rdf+flavour.rdf).
