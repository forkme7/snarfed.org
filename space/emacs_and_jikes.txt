Using
[Emacs](http://gnu.org/software/emacs/) and the
[Jikes](http://www.ibm.com/developerWorks/oss/jikes/) JDK? Want error messages
in a format that Emacs can parse, so you can use **C-x** \` (ie the
**next-error** function)? Run jikes with the **+D** argument.
